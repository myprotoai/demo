## ğŸ› ï¸ Hardware Requirements

### Main Components
- **STM32U083KCU6** - Ultra-low-power microcontroller
- **SGP40-D-R4** - Professional TVOC air quality sensor
- **MCP73832T** - Li-Po battery charge controller
- **ADP150AUJZ-2.5-R7** - Low-dropout voltage regulator
- **EAST1616RGBA1** - RGB LEDs (4x)
- **PKMCS0909E4000-R1** - Piezoelectric buzzer

### Tools Required
- Soldering iron and solder
- ST-Link V2 programmer (for firmware upload)
- 3D printer (for enclosure)
- Basic electronics tools

## ğŸš€ About ProtoAI

ProtoAI transforms hardware sketches into manufacturable products using AI. Generates 2D diagrams, BOMs, assembly instructions, 3D models (STL), PCB files, and firmware.

**Note**: The ProtoAI platform source code is not included in this repository. This repo contains sample outputs (assembly instructions, 3D models, PCB files, firmware) generated by ProtoAI to demonstrate its capabilities.
